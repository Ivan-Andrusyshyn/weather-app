# Weather App

## Description

**Weather Diary App** is a web application built with **Angular** and **Tailwind CSS** that allows users to view the current weather in any city.

The app integrates with the [OpenWeather API](https://api.openweathermap.org) for weather data.

-------

## Technologies

- **Frontend:** Angular 17.2  
- **Styling:** Tailwind CSS, responsive (mobile-first) design    
- **Weather API:** [OpenWeather API](https://api.openweathermap.org)
